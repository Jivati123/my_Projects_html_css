For Each Loop

  • forEach() is only valid for the Arrays and not strings.
  • forEach() loop is called as a HOF (Higher Order Function / Method)
  • The forEach() method calls a function for each element in an array.
  • SO BASICALLY forEach () IS A FUNCTION WHICH USES ANY OTHER FUNCTION AS AN PARAMERER TO RETURNS A FUNCTION AS AN ARGUMENT. 
  •             Syntax:
            array.forEach(function(currentValue, index, arr), thisValue)
        
  •             Example 1
            Calls a function for each element in fruits:
            
            const fruits = ["apple", "orange", "cherry"];
            fruits.forEach(myFunction);
